# Recipe Generator Project

## Descripción
El **Generador de Recetas** es una aplicación web diseñada para ayudar a los usuarios a crear recetas personalizadas utilizando inteligencia artificial. La aplicación permite a los usuarios ingresar ingredientes y recibir recetas adaptadas a sus preferencias y necesidades dietéticas.

## Estructura del Proyecto
El proyecto está organizado en varias carpetas y archivos, cada uno con un propósito específico:

- **.github/skills/**: Contiene habilidades específicas para la arquitectura del proyecto.
- **docs/specs/**: Documentación que incluye especificaciones, diseño y tareas relacionadas con el generador de recetas.
- **src/**: Código fuente de la aplicación, incluyendo componentes, servicios y modelos de datos.
- **AGENTS.md**: Documento que describe el contexto y las reglas del proyecto.
- **package.json**: Metadatos del proyecto, incluyendo dependencias y scripts.
- **tsconfig.json**: Configuración de TypeScript.
- **vite.config.ts**: Configuración de Vite, la herramienta de construcción utilizada en el proyecto.

## Instalación
Para instalar las dependencias del proyecto, ejecuta el siguiente comando en la raíz del proyecto:

```bash
npm install
```

## Desarrollo
Para iniciar el entorno de desarrollo, utiliza el siguiente comando:

```bash
npm run dev
```

Esto iniciará la aplicación en modo de desarrollo y estará disponible en `http://localhost:3000`.

## Construcción
Para crear una versión de producción de la aplicación, ejecuta:

```bash
npm run build
```

## Preview
Para ver la versión de producción de la aplicación, utiliza:

```bash
npm run serve
```
